# 🎓 Student Performance Prediction System

## 📌 Project Overview
This project predicts student academic performance using Machine Learning. It analyzes factors like study time, attendance, and previous marks to identify students who are at risk of failing.

---

## 🎯 Objective
- Predict student performance (Pass/Fail)
- Identify at-risk students early
- Help improve academic outcomes using data-driven insights

---

## 📊 Dataset
- Source: UCI Machine Learning Repository
- File: `student-mat.csv`
- Total Records: 395 students
- Features include:
  - Study Time
  - Absences
  - Previous Marks (G1, G2)
  - Final Marks (G3)

---

## ⚙️ Technologies Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn
- Google Colab

---

## 🔍 Methodology
1. Data Collection
2. Data Preprocessing (handling, encoding)
3. Exploratory Data Analysis (EDA)
4. Feature Selection
5. Model Training
6. Model Evaluation

---

## 🤖 Machine Learning Models Used
- Logistic Regression
- Decision Tree
- Naive Bayes
- Support Vector Machine (SVM)

---

## 📈 Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score
- Confusion Matrix

---

## ⚠️ At-Risk Student Identification
Students predicted as **Fail (0)** are classified as **At-Risk Students**.  
This helps institutions take early corrective actions.

---

## 📊 Output
- Student performance prediction
- Confusion matrix visualization
- Model comparison results
- List of at-risk students

---

## 📌 Results
The models achieved good accuracy (not 100%), showing realistic performance on real-world data. Logistic Regression and SVM performed consistently well.
